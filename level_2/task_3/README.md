# Task Manager CRUD Console Application

## Objective
To implement Create, Read, Update, and Delete operations using a list for data storage.

## Description
This project is a console-based Task Manager application developed using Java.
It allows the user to create new tasks, display all tasks, update existing tasks, and delete tasks.

## Technologies Used
- Java

## Concepts Used
- Class and objects
- ArrayList
- Functions
- Loops
- Conditional statements
- CRUD operations

## How to Run
```bash
javac TaskManager.java
java TaskManager
```
