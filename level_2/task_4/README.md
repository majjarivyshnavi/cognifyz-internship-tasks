# Temperature Converter Program

## Objective
To create a temperature converter program that converts temperatures between Celsius and Fahrenheit.

## Description
This project is a console-based temperature converter program developed using Java.
It allows the user to choose the conversion direction between Celsius to Fahrenheit and Fahrenheit to Celsius.

## Technologies Used
- Java

## Concepts Used
- Functions
- Input and output
- Conditional statements
- Loops
- Arithmetic operations

## How to Run
```bash
javac TemperatureConverter.java
java TemperatureConverter
```
