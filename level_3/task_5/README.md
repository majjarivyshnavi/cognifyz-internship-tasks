# Task Manager with File Storage

## Objective
To enhance the CRUD task manager application by storing task data persistently using file I/O.

## Description
This project is a console-based Task Manager application developed using Java.
It allows users to create, display, update, and delete tasks.
The task data is stored in a text file named `tasks.txt`, so data remains available even after the program is closed.

## Technologies Used
- Java

## Concepts Used
- Class and objects
- ArrayList
- File handling
- CRUD operations
- Error handling

## How to Run
```bash
javac TaskManagerFileIO.java
java TaskManagerFileIO
```
