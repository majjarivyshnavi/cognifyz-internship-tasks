# Interactive Web Scraper

## Objective
To fetch data from a website and present it in a user-friendly format using a simple web scraping approach.

## Description
This project is a simple web scraping program developed using Java.
It fetches quotes and author names from a website and displays them clearly in the console.

## Website Used
https://quotes.toscrape.com/

## Technologies Used
- Java

## Concepts Used
- Web scraping
- HTTP connection
- String processing
- Pattern matching
- Error handling

## How to Run
```bash
javac WebScraper.java
java WebScraper
```
