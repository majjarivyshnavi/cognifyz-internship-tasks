# Number Guessing Game

## Objective
To develop a basic text-based game using conditional statements for game logic.

## Description
This is a simple text-based Number Guessing Game developed using Java.
The computer randomly selects a number between 1 and 10.
The user has 3 chances to guess the correct number.

## Technologies Used
- Java

## Concepts Used
- Conditional statements
- Loops
- Random class
- Scanner class
- Input and output

## How to Run
```bash
javac NumberGuessingGame.java
java NumberGuessingGame
```
