# Number Pattern Generator

## Objective
To generate and print simple number patterns using loops.

## Description
This project generates a simple number pyramid pattern using Java.
The user enters the number of rows, and the program prints the pattern using nested loops.

## Technologies Used
- Java

## Concepts Used
- For loop
- Nested loop
- Scanner class
- Input and output

## How to Run
```bash
javac NumberPattern.java
java NumberPattern
```
